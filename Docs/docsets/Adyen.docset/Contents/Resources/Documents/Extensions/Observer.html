Observer

public protocol Observer : AnyObject
  • observe(_:eventHandler:)

    Observes an event publisher for events.

    Declaration

    Swift

    @discardableResult
func observe<T: EventPublisher>(_ eventPublisher: T, eventHandler: @escaping EventHandler<T.Event>) -> Observation

    Parameters

    eventPublisher

    The event publisher to observe.
    eventHandler

    A handler to invoke for new events.

    Return Value

    An observation, representing the created observation. Can be used to remove the observation later.

  • bind(_:to:at:)

    Binds the value of an observable to a key path. It will also set the current value of the observable to the key path.

    Declaration

    Swift

    @discardableResult
func bind<Value, Target>(_ observable: Observable<Value>, to target: Target, at keyPath: ReferenceWritableKeyPath<Target, Value>) -> Observation where Value : Equatable, Target : AnyObject

    Parameters

    observable

    The observable to observe for new values.
    target

    The target of where to set the new values.
    keyPath

    The key path to set the new values.

    Return Value

    An observation that represents the binding. Can be used to remove the binding later.
